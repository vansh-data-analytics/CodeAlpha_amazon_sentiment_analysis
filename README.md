# Amazon Review Sentiment Analysis

## Project Overview
This project performs Sentiment Analysis on Amazon customer reviews using Python and Natural Language Processing (NLP). The reviews are classified into three categories:

- Positive
- Negative
- Neutral

The project helps understand customer opinions and overall product satisfaction through data analysis and visualization.

---

# Technologies Used

- Python
- Google Colab
- Pandas
- TextBlob
- Matplotlib
- Seaborn

---

# Dataset Information

The dataset contains Amazon customer reviews with details such as:

- Reviewer Name
- Rating
- Review Title
- Review Text
- Review Date
- Country

---

# Project Workflow

1. Dataset Collection
2. Data Loading using Pandas
3. Data Cleaning
4. Sentiment Classification using TextBlob
5. Data Visualization
6. Insights Generation
7. Export Final Output CSV

---

# Sentiment Categories

| Sentiment | Meaning |
|---|---|
| Positive | Happy/Satisfied customer reviews |
| Negative | Unhappy/Unsatisfied customer reviews |
| Neutral | Normal or mixed reviews |

---

# Key Insights

- Most customer reviews were identified as Positive.
- Negative reviews mainly reflected issues related to product quality, delivery, or customer service.
- Neutral reviews contained mixed or unclear opinions.
- Sentiment analysis helps businesses understand customer satisfaction levels.
- Visualization made it easier to compare customer opinions.

---

# Visualization

The project includes graphical visualization of sentiments using:

- Count Plot
